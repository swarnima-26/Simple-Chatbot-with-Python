# Simple Chatbot using Python

## 1. Project Description

This project is a simple chatbot developed using Python. The chatbot interacts with users through text input and provides responses based on predefined keywords.

It is a rule-based chatbot, which means it does not use advanced machine learning but follows simple logic to understand user input and generate replies.

This project is designed for beginners to understand the basic working of chatbots and human-computer interaction.


## 2. Features

* Responds to greetings like "hi", "hello"
* Provides current time and date
* Answers basic queries related to college and courses
* Continuous conversation until user exits
* Simple and easy-to-understand code


## 3. Technologies Used

* Python 3
* Built-in libraries:

  * random
  * datetime

No external libraries are required.

## 5. How to Set Up the Project

### Step 1: Install Python

Download and install Python from the official website:
https://www.python.org/

Make sure Python is added to PATH.


### Step 2: Download the Project

You can either:

* Clone the repository:
  git clone <your-repo-link>

OR

* Download the ZIP file and extract it


### Step 3: Open Project Folder

Open the folder in Command Prompt or Terminal.


## 6. How to Run the Program

Run the following command:

python chatbot.py


## 7. How to Use the Chatbot

After running the program:

1. The chatbot will start and display a message
2. Type your message and press Enter
3. The chatbot will respond
4. Continue chatting
5. Type "bye" or "exit" to stop the program


## 8. Example Interaction

You: hello
Chatbot: Hello!

You: what is the time
Chatbot: Current time is 12:30:00

You: bye
Chatbot: Goodbye

## 10. Limitations

* Works only on predefined keywords
* Cannot understand complex sentences
* Does not learn from new inputs
